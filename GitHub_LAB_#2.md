# **Лабораторная работа #2**

### **Задание по программированию: «Указатели и ссылки»**
Напишите программу на C++, которая:
- Объявляет переменную `int x = 100`;
- Создаёт указатель `ptr` и ссылку `ref` на эту переменную;
- Выводит с помощью `cout`:
  - значение `x`, `*ptr`, `ref`;
  - адрес `x`, значение `ptr`, адрес `ref`.

Пример вывода:
```
Значения: x = 100, *ptr = 100, ref = 100
Адреса: &x = 0x..., ptr = 0x..., &ref = 0x...
```

Сохраните программу в файл: `main.cpp`.

---

### **Шаги по работе с Git**

1. **Создайте репозиторий на GitHub**  
   - Зайдите на [github.com](https://github.com)
   - Нажмите **«New repository»**
   - Название: `cpp-pointers-lab`
   - Поставьте галочку **«Add a README file»**
   - **Не** добавляйте `.gitignore` — вы создадите его сами!

2. **Склонируйте репозиторий на компьютер**  
   Откройте **Git Bash** в удобной папке и выполните:
   ```bash
   git clone https://github.com/ваш-логин/cpp-pointers-lab.git
   ```

3. **Добавьте файл `.gitignore`**  
   Создайте файл `.gitignore` и добавьте в него строки для Visual Studio:
   ```
   *.suo
   *.user
   *.vcxproj.filters
   Debug/
   Release/
   *.sdf
   *.opensdf
   ```

4. **Поместите ваш `main.cpp` в папку проекта**  
   Убедитесь, что файл лежит в той же папке, что и `README.md`.

5. **Сделайте первый коммит**  
   Выполните в Git Bash:
   ```bash
   git add .
   git commit -m "Добавлена программа по указателям и ссылкам"
   ```

6. **Отправьте на GitHub**  
   ```bash
   git push
   ```

> ✅ Готово! Теперь ваш код хранится в удалённом репозитории, и вы сделали свой первый настоящий коммит.

---

### **Что проверить на GitHub после пуша?**
- В репозитории должны быть файлы: `README.md`, `.gitignore`, `main.cpp`.
- В истории коммитов — ваш коммит с описанием.
