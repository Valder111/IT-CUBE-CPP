# **Краткая теория по начальному C++**

## **1. Переменные и типы данных**

В C++ каждая переменная должна быть объявлена с указанием её типа. Основные встроенные типы данных:

| Тип | Описание | Пример |
|-----|---------|--------|
| `int` | Целое число | `int age = 18;` |
| `double` | Вещественное число с плавающей точкой (высокая точность) | `double price = 199.99;` |
| `float` | Вещественное число с плавающей точкой (меньше точность, чем `double`) | `float weight = 75.5f;` |
| `char` | Символ | `char letter = 'A';` |
| `bool` | Логическое значение (`true` / `false`) | `bool isActive = true;` |
| `string` | Строка (требует подключения `<string>`) | `string name = "Ivan";` |

Объявление переменной:  
```cpp
тип имя_переменной = значение;
```

Имена переменных должны начинаться с буквы или подчеркивания, не содержать пробелов и не совпадать с ключевыми словами.

## **2. Базовый синтаксис C++**

Любая программа на C++ начинается с функции `main()`. Минимальный шаблон программы:

```cpp
#include <iostream>     // подключение стандартной библиотеки ввода-вывода
using namespace std;    // упрощение записи (не обязательно, но удобно для обучения)

int main() {
    cout << "Hello, World!" << endl;  // вывод на экран
    return 0;                         // завершение программы (успешно)
}
```

- `#include <...>` — директива препроцессора для подключения библиотек.
- `using namespace std;` — позволяет использовать `cout`, `cin` без префикса `std::`.
- `int main()` — точка входа в программу.
- `cout` — вывод в консоль.
- `cin` — ввод с клавиатуры.
- `return 0;` — программа завершилась без ошибок.

Пример ввода:
```cpp
int x;
cin >> x;
```

## **3. Массивы**

### **Одномерные массивы**
Фиксированный набор элементов одного типа.

Объявление:
```cpp
тип имя[размер];
```

Пример:
```cpp
int arr[5];               // массив из 5 целых чисел (индексы 0–4)
int numbers[] = {1, 2, 3}; // автоматический размер по числу элементов
```

### **Двумерные массивы**
Массив массивов (таблица).

Пример:
```cpp
int matrix[3][3];         // квадратная матрица 3×3
int table[2][4] = {
    {1, 2, 3, 4},
    {5, 6, 7, 8}
};
```

### **Векторы (`vector`)**
Динамический массив из стандартной библиотеки (`<vector>`). Размер можно менять.

Подключение:
```cpp
#include <vector>
using namespace std;
```

Примеры:
```cpp
vector<int> v;            // пустой вектор целых чисел
vector<int> v2 = {1, 2, 3}; // инициализация
v.push_back(10);          // добавление элемента в конец
v.size();                 // количество элементов
v[0];                     // доступ к элементу (без проверки границ!)
```

Преимущества `vector`: автоматическое управление памятью, гибкий размер, удобные методы.

## **4. Функции и процедуры**

В C++ нет отдельного понятия "процедура" — все подпрограммы называются **функциями**. Функция может возвращать значение (`return`) или не возвращать (тип `void`).

### **Объявление и определение**
```cpp
тип_возврата имя_функции(список_параметров) {
    // тело функции
}
```

Примеры:
```cpp
// Функция, возвращающая сумму
int add(int a, int b) {
    return a + b;
}

// Процедура (ничего не возвращает)
void greet(string name) {
    cout << "Hello, " << name << "!" << endl;
}
```

### **Передача параметров**
- **По значению**: `void func(int x)` — создаётся копия.
- **По ссылке**: `void func(int &x)` — изменения влияют на оригинал.
- **По константной ссылке**: `void func(const int &x)` — безопасно и эффективно для больших объектов.

## **5. Работа с файлами**

Для работы с файлами используются классы из `<fstream>`:
- `ifstream` — чтение из файла.
- `ofstream` — запись в файл.
- `fstream` — чтение и запись.

### **Чтение из файла**
```cpp
#include <fstream>
#include <iostream>
using namespace std;

ifstream fin("input.txt");
int x;
if (fin.is_open()) {
    fin >> x;             // чтение числа
    fin.close();
}
```

### **Запись в файл**
```cpp
ofstream fout("output.txt");
if (fout.is_open()) {
    fout << "Result: " << x << endl;
    fout.close();
}
```

### **Проверка успешности открытия**
Всегда рекомендуется проверять `is_open()` или использовать файловый поток в условии:
```cpp
if (fin) { /* файл открыт успешно */ }
```

Можно читать построчно:
```cpp
string line;
while (getline(fin, line)) {
    cout << line << endl;
}
```

---

> **Примечание**: Все примеры предполагают использование компилятора C++11 или новее (для `vector` инициализации и т.д.).
