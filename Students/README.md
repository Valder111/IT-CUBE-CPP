# Рабочая директория для ответов обучающихся

Сюда вы заливаете свои работы. Структура должна выглядить следующим образом:

* Ваша личная папка: `Ivanov_Ivan`
  * Внутри вы создаёте папки для каждой отдельной выполненной работы, а название вы им даёте в зависимости от характера работы: `laba#1` , `sortirovka_CPP` и т.д.

Все обновления в этот репозиторий отправляются и принимаются в формате Pull Request (после проверки преподавателем).
