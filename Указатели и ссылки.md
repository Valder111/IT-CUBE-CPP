# Указатели и ссылки в С++
---
### 1. **Что такое указатель (`*`)?**

**Указатель** — это переменная, которая хранит **адрес другой переменной** в памяти.

- Объявление:  
  ```cpp
  int a = 10;
  int* p = &a;  // p — указатель на переменную a
  ```
- Оператор `&` — получает **адрес** переменной.  
- Оператор `*` — **разыменовывает** указатель, то есть получает значение по адресу:
  
  ```cpp
  cout << *p;  // выведет 10
  ```

**Особенности:**
- Указатель может быть **изменён** — можно заставить его указывать на другую переменную.
- Указатель может быть **равен `nullptr`** (ничего не указывает).
- Указатели позволяют **изменять переменные вне функции**, передавая их по адресу.

---

### 2. **Что такое ссылка (`&`)?**

**Ссылка** — это **псевдоним** (другое имя) для уже существующей переменной.

- Объявление:
  ```cpp
  int a = 10;
  int& r = a;  // r — ссылка на a
  ```
- Теперь `r` и `a` — это **одно и то же** в памяти.
- Изменение `r` изменяет `a`, и наоборот.

**Особенности:**
- Ссылка **не может быть "пустой"** — она всегда привязана к какой-то переменной.
- Ссылку **нельзя переопределить** — она не может "перепрыгнуть" на другую переменную.
- Удобна для **передачи в функции**, чтобы избежать копирования и работать с оригиналом.

---

## Пример:

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 42;

    // Указатель на a
    int* ptr = &a;

    // Ссылка на a
    int& ref = a;

    cout << "Значение a: "        << a     << endl;
    cout << "Значение через *ptr: " << *ptr  << endl;
    cout << "Значение через ref: "  << ref   << endl;

    cout << "Адрес a: "           << &a    << endl;
    cout << "Адрес, хранящийся в ptr: " << ptr  << endl;
    cout << "Адрес ref (на самом деле адрес a): " << &ref << endl;

    return 0;
}
```

### Что будет выведено (примерно):
```
Значение a: 42
Значение через *ptr: 42
Значение через ref: 42
Адрес a: 0x7fff5fbff6ac
Адрес, хранящийся в ptr: 0x7fff5fbff6ac
Адрес ref (на самом деле адрес a): 0x7fff5fbff6ac
```
### Вывод:
- `*ptr` и `ref` ведут себя как сама переменная `a`.
- Указатель `ptr` хранит адрес, и чтобы получить значение — нужен `*`.
- Ссылка `ref` — это просто другое имя для `a`, работает напрямую.
- Адреса `&a`, `ptr` и `&ref` совпадают — все ссылаются на одно место в памяти.
